התקדמות:Progress:
28
כל הספרים בתנ"ך מוכנים בגליון הנתונים הזה, וכבר הועלו פרקיהם במלואם בוויקיטקסט. בוויקיטקסט עדיין נשארות השלמה ועריכה סופית לפרקי המבוא השלם למהדורה.All of the books of Tanakh in this spreadsheet are complete, and their chapters have been uploaded to Wikisource. At Wikisource the chapters of the full introduction to this edition (in Hebrew) still need to be completed and undergo final editing.
29
הערה: גליון הנתונים הזה מכיל את הנוסח המלא (כולל תיעוד מקומי) לכל הפרקים והפסוקים שבמקרא, וגם רשימה מבוארת של התבניות המופיעות בתוך הטקסט. הוא *אינו* מכיל את ההקדמה המתודולוגית המלאה למהדורה זו, את המדריך הטכני לגליון הנתונים, את הקוד לתבניות, ואת הקוד ליצירת דפים מיוחדים לפרשות השבוע. כל החומר ההוא נגיש באתר ויקיטקסט (קישורים למעלה).Note: This spreadsheet contains the full text (including local documentation) for all of the chapters and verses in the Hebrew Bible, as well as an explanatory list of the templates which appear inside the base text. It does *not* include the overall introduction that explains this edition's methodology, the technical guide for the spreadsheet, the code for templates, the code for pages showing the weekly Torah portions, nor the coded text for biblical passages which this edition shows in special formats. All of that material is available at Wikisource (links appear above).
